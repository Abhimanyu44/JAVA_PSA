# Access Specifier / Access Modifier:

                                        Private        Default        Protected        Public 
-----------------------------------------------------------------------------------------------
- same class                            yes
-----------------------------------------------------------------------------------------------
- same package sub class                no 
-----------------------------------------------------------------------------------------------
- same package non sub class            no
-----------------------------------------------------------------------------------------------    
- different package sub class           no
-----------------------------------------------------------------------------------------------
- different package non sub class       no
-----------------------------------------------------------------------------------------------

#Note:
    - If we make a 'method' or 'variable' 'private' then it can can be accessed only inside 'same class' but not outside the class.
